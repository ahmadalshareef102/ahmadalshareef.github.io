الأسئلة

👥 استيراد قائمة الطلاب

الصق قائمة الطلاب بأحد الأشكال التالية

الصيغة المقبولة: كل طالب على سطر منفصل، والاسم ثم الرقم مفصولين بفاصلة أو مسافة أو tab.

أمثلة: محمد علي حسن الأهدل, 202312101 فاطمة أحمد الحكيمي, 202312102 عبدالله سعيد الزبيدي, 202312103 سارة عبدالله المخلافي, 202312104

📥 استيراد الأسئلة

اختر طريقة الاستيراد المناسبة لك

الصيغة: كل سؤال على سطر منفصل. ابدأ بـ Q: للسؤال، ثم كل خيار على سطر يبدأ بـ A) أو B)، والإجابة الصحيحة ضع أمامها *.

مثال:
Q: ما عاصمة اليمن؟ A) عدن *B) صنعاء C) تعز D) الحديدة Q: 2 + 2 = ؟ A) 3 *B) 4 C) 5
صيغة JSON احترافية مع دعم كامل لجميع أنواع الأسئلة.

types: mcq, tf, fill, essay
ارفع ملف نصي (.txt) أو ملف JSON (.json). يمكنك أيضاً رفع ملف CSV بصيغة: النوع,السؤال,الخيارات,الإجابة,الدرجة

👤 بيانات المدرس

البريد المرتبط بحسابك في Firebase Authentication
تُغيَّر من Firebase Console أو "نسيت كلمة السر" من شاشة الدخول
🔐 النظام الجديد: تسجيل الدخول محمي بـ Firebase Authentication. كلمة السر مشفّرة ومحفوظة بأمان لدى Google ولن تظهر في قاعدة البيانات.

🎨 الواجهة والذكاء المساند

يمكنك وضع ملف الخط في المسار fonts/tttr.ttf ليعمل تلقائياً.
يُستخدم فقط إذا اخترت API خارجي.
التثبيت يعمل عند تشغيل المنصة عبر HTTPS أو Firebase Hosting، وليس من file:// مباشرة.

☁️ الربط مع Firebase

الحالة: محلي فقط
هذه النسخة تدعم مزامنة الاختبارات والنتائج والمراقبة الحية بين الأجهزة عبر Firestore. الصق بيانات مشروع Firebase Web App ثم احفظ واختبر الاتصال.
يمكن تغييره إذا أردت أكثر من منصة داخل نفس المشروع
